# warehouse_manager

## Features
- Create an account with email and password(employer only)
- Login with email and password
- Create order
- Change status order (admin only)
- All customer orders (client)
- All orders (admin only)
- Add comments (owner of the order & admin)

## Technologies
- Python3/Django
- React
- Webpack
